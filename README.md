# Starter template
Starter template for projects running on Bootstrap, SASS, ES6.

## Installation

1. `npm install`
2. `gulp watch`
3. In your browser open localhost:3000/starter
   
## Customization

Working directory is `starter/src`. Resulting files are compiled into the `starter/assets` directory.

## Gulp commands

* `gulp watch`
* `gulp default`
* `gulp vendor`
* `gulp css`
* `gulp js`

~ Happy coding!
